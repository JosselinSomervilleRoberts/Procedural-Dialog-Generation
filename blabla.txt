LES COMMANDES DE BASES AVEC GITHUB :
- Accéder à l'invite de commande git : clic-droit, "Git Bash Here"
- "git pull" avant de commencer à coder, pour actualiser ses fichiers locaux
- Pour mettre ses fichiers sur le commun, "git add ."
  Puis "git commit -m "[type de modification][ton nom] libellé""
  Puis "git push" (si erreur, faire "git pull", puis "git push")
